How to Play

  1. Select the difficulty level:
    • Easy: Longer time between flashes.
    • Medium: Moderate time between flashes.
    • Hard: Shorter time between flashes.
  2. Press any key to start the game.
  3. At the beginning of the game, one of the 4 keys lights up randomly producing.
  4. The player has to press the same key.
  5. Next, Simon turns back the same light on and a second one, again randomly.
  6. The player has to reproduce this chain of light using their memory.
